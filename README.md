# AR Heritage Platform

An Augmented Reality (AR) platform developed for the Smart India Hackathon (SIH) to provide an immersive and interactive way of exploring cultural heritage sites and historical monuments.

## Overview

AR Heritage Platform enables users to experience heritage locations through augmented reality, interactive 3D models, and educational content. The application combines AR technology with cultural heritage preservation to make learning more engaging and accessible.

## Features

* Augmented Reality-based heritage visualization
* Interactive 3D models of historical monuments
* Educational information and cultural insights
* Mobile-friendly user experience
* Real-time AR object placement and interaction
* Heritage site exploration and learning

## Technologies Used

* Unity
* AR Foundation
* Vuforia Engine
* C#
* Android Platform
* 3D Models (FBX)
* TextMesh Pro

## Requirements

* Unity 6.x (or the Unity version used for development)
* Android SDK
* AR-supported Android device
* Vuforia Engine Package

## Installation

1. Clone the repository:

```bash
git clone https://github.com/Bharathasi59/Ar-heritage-Platform.git
```

2. Open the project in Unity.

3. Install required packages if prompted.

4. Configure Vuforia License Key.

5. Build and run on an AR-supported Android device.
